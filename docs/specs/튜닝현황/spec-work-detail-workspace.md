# 작업 상세 — 워크스페이스 (T2)

## 화면 목적

슬라이드 패널(T1 Peek)에서 깊이 있는 검토가 필요할 때 전환하는 전체화면 모드. SQL/Plan을 항상 좌우 병렬 Diff로 비교하며, AI 분석 근거를 독립 패널로 분리하여 코드 중심의 꼼꼼한 리뷰를 지원한다.

> **현재 구현 상태** *(2026-04-24 기준)*: 워크스페이스 진입·좌우 기본 비교 레이아웃은 동작. **스코프 바(SQL/Plan/All/Bind), SQL Diff / Plan Diff 토글 버튼, Describe Object 패널, 전체보기(Maximize2) 버튼, 수평/수직 드래그 분할선**은 DOM 에 미구현 — 구현 예정 기획. 현재 기본 Before/After 좌우 병렬 렌더만 노출됨.

### 3-Tier Progressive Disclosure

| Tier | 이름 | 너비 | 용도 | 진입 |
|------|------|------|------|------|
| T1 | Peek (슬라이드) | 40% 뷰포트 | 빠른 트리아지 | 행 클릭 |
| **T2** | **워크스페이스** | **100% (최대화)** | **꼼꼼 분석** | **F키 / 최대화 버튼 / 상세검토 버튼** |
| T3 | Detached (새 탭) | 별도 브라우저 탭 | 병렬 참조 | 새 탭 버튼 |

### 사용자 시나리오

**시나리오 1 — 빠른 검토 → 깊은 분석 전환**
작업함에서 행 클릭 → T1 슬라이드에서 메트릭 확인 → 개선율이 애매하여 SQL Diff 상세 확인 필요 → F키 또는 상세검토 버튼 → T2 워크스페이스에서 좌우 병렬 Diff로 꼼꼼 비교 → 확인/반려 판단

**시나리오 2 — Plan 기반 판단**
T2 진입 → AI 분석 패널에서 근거 확인 → 우측 Plan Diff에서 Full Table Scan → Index Range Scan 전환 확인 → 인덱스 전략 타당성 판단 → 확인

### (0) 워크스페이스 헤더 + 작업 목록

**1. 내용**
- 워크스페이스 상단 헤더와 좌측 작업 목록 사이드바를 제공한다.

**2. 구성**
- a. 헤더 바
 - "← 목록" 버튼 — 워크스페이스 닫기, 목록 화면 복귀
 - SQL ID — 고정폭 텍스트 (SQL 입력 건이면 "SQL 입력" 표시)
 - 상태 인디케이터 — StatusIndicator 컴포넌트 (CostSweepHero 애니메이션 포함)
 - 인스턴스명
 - 우측: "+ 새 입력" 버튼 (SQL 입력 화면 진입), 새 탭 아이콘 (ExternalLink)
- b. 탭 바 (튜닝 데이터 있을 때만)
 - 검토 | 반영 | 운영효과 | 이력
 - 선택된 탭: 파란 텍스트 + 하단 2px 파란 보더
- c. 작업 목록 사이드바 (좌측)
 - "작업 목록 N" 헤더 + 접기 버튼 (ChevronLeft)
 - 스크롤 가능한 작업 카드 목록
 - 각 카드: 배치명 · 인스턴스 · 개선율, SQL ID, SQL 미리보기 텍스트
 - 선택된 카드: 파란 좌측 스트라이프
 - 하단: "SQL 입력" 버튼

**3. 동작**
- a. 목록 카드 클릭 → 해당 작업으로 워크스페이스 콘텐츠 전환

### (1) 상세 정보 패널 (좌측)

**1. 내용**
- 검토 탭의 메타/메트릭/AI근거/바인드변수/액션을 좌측 독립 패널에 세로 배치한다.

**2. 구성**
- a. 패널 헤더
 - "상세 정보" 제목 — 굵은 텍스트
 - 접기 버튼 (ChevronLeft 아이콘) — 우측 정렬
- b. 메타 정보
 - 인스턴스명 · 스키마명 · 날짜 (한 줄)
 - 출처 배지 — MaxGauge / AWR / V$SQL (배경+텍스트 색상)
 - 실행검증 배지 — 실측(초록) / 실측(T/O)(노랑) / 예상(회색)
 - 정합성 경고 — 불일치 시 "⚠ 불일치" 빨간 배지
- c. 성능 메트릭 카드 — 테두리 라운드 카드, 2열 구분선
 - Elapsed — Before 값(취소선) → After 값(굵은), 개선율(↓N%, 초록/빨강)
 - Buffer Gets — 동일 형식
- d. AI 분석 근거 (→ 검토 탭 (4)와 동일)
 - 요약 1~2줄
 - 접기/펼치기 (기본 접힘)
 - 상세 근거 목록 (인덱스 전략, 힌트 근거 등)
- e. 바인드 변수 (→ 검토 탭 (5)와 동일)
 - 변수 테이블 — 변수명 / 타입 / 값
 - 캡처 시점
 - 복사 버튼
 - "바인드 검증" 버튼 — 수집된 세트가 여러 개일 때 표시
- f. 액션 버튼 — **패널 하단 sticky 고정** (스크롤과 무관하게 항상 보임), 흰색 배경, 상단 1px 구분선

 - "승인" 버튼 — 전체 너비, CheckCircle2 아이콘, primary
 - "재튜닝" 버튼 — 좌측 절반, Undo2 아이콘, secondary
 - "반려" 버튼 — 우측 절반, XCircle 아이콘, danger
 - 튜닝완료(approval_pending) 상태일 때만 표시
- g. 리사이즈 핸들 — 우측 경계 4px, 드래그로 패널 너비 조절, 호버/드래그 시 파란 하이라이트

### (2) SQL/Plan 비교 영역 (우측) — WorkbenchCompareView

**1. 내용**
- Before/After SQL과 실행계획을 스코프 전환 방식으로 비교한다. 기본 All 모드에서 4패널(SQL Before/After + Plan Before/After)을 동시 표시.

**2. 구성**
- a. 스코프 바 (상단 고정, 배경 구분)
 - 스코프 버튼 2그룹:
 - 1그룹: `SQL` | `Plan` | `All` — SQL만/Plan만/전체 비교 (기본값 All)
 - 2그룹: `Bind` — 바인드 변수 데이터가 있을 때만 표시. 바인드 검증 피벗 테이블 + 실행결과 비교
 - 선택된 스코프 — 파란 배경+흰 글씨, 미선택 — 회색
 - 전체보기 라벨 — 개별 패널 전체보기 진입 시 "SQL — Before" 등 라벨 + X 닫기 버튼 표시
 - Diff 범례 — SQL Diff 또는 Plan Diff 켜져 있을 때 표시: 변경(노랑) · 추가(초록) · 삭제(빨강) 색상 칩
 - 우측 컨트롤:
 - `Format` 버튼 — SQL 구문 포매팅 on/off. 켜짐 시 파란 배경
 - `SQL Diff` 버튼 — SQL 변경사항 인라인 하이라이트. 켜짐 시 파란 배경
 - `Plan Diff` 버튼 — Plan 변경사항 하이라이트. 켜짐 시 파란 배경
- b. SQL 비교 섹션 (상단)
 - "SQL" 섹션 라벨 — 좌측 세로 바 + 대문자 타이틀
 - Before 패널 (좌)
 - 헤더: "Before" 라벨 + 세로 바 구분
 - 우측 버튼: 복사 버튼(Copy 아이콘) + 전체보기 버튼(Maximize2 아이콘)
 - 본문: 고정폭 SQL 텍스트, 키워드 구문 강조
 - Diff 모드: 줄 단위 배경색 — 삭제(빨강), 변경(노랑)
 - After 패널 (우) — 동일 구성
 - Diff 모드: 추가(초록), 변경(노랑)
- c. 수평 드래그 분할선 — SQL/Plan 영역 높이 비율 조절 (All 스코프)
 - 분할선: 높이 8px, 중앙에 둥근 핸들, 호버 시 강조
 - 드래그 범위: 10%~80%
 - 기본값: 40:60 (SQL:Plan)
- e. 수직 드래그 분할선 — Before/After 좌우 비율 조절 (SQL, Plan 스코프)
 - 분할선: 너비 8px, 중앙에 둥근 핸들, 호버 시 강조
 - 드래그 범위: 20%~80%
 - 기본값: 50:50 (Before:After)
- d. Plan 비교 섹션 (하단)
 - "PLAN" 섹션 라벨 — 좌측 세로 바 + 대문자 타이틀
 - Before 패널 (좌)
 - 헤더: "Before" 라벨 + 복사 + 전체보기
 - 본문: 실행계획 테이블 (Id, Operation, Name, Starts, E-Rows, A-Rows 등)
 - Diff 모드: 변경된 Operation 행 배경색 강조
 - After 패널 (우) — 동일 구성

**3. 동작**
- a. 스코프 전환
 - `SQL` → SQL Before/After만 좌우 표시. Plan 숨김. 수직 분할선으로 Before/After 비율 조절 가능
 - `Plan` → Plan Before/After만 좌우 표시. SQL 숨김. 수직 분할선으로 Before/After 비율 조절 가능
 - `All` → SQL Before/After(상단) + Plan Before/After(하단) 4패널 전체 표시. 수평 분할선으로 SQL/Plan 비율 조절
 - `Bind` → 바인드 검증 피벗 테이블 (행=변수, 열=캡처시점) + Before/After 실행결과 비교. 바인드 데이터가 있을 때만 스코프 버튼 표시
- b. 전체보기 클릭 (Maximize2 아이콘) → 해당 패널만 전체 영역에 확대 표시. 스코프 바에 "SQL — Before" 라벨 + X 닫기 버튼 추가. X 클릭 → 이전 스코프로 복귀
- c. 복사 클릭 → 해당 패널의 텍스트 클립보드 복사. 아이콘이 체크마크(✓)로 1.5초 전환 후 원복
- d. Format 토글 → SQL 들여쓰기 적용/해제. SQL이 표시될 때만 활성
- e. SQL Diff 토글 → SQL 변경사항 줄 단위 배경색 하이라이트 on/off. 전체보기 모드가 아닐 때만 표시
- f. Plan Diff 토글 → Plan 변경사항 행 단위 하이라이트 on/off. 전체보기 모드가 아닐 때만 표시
- g. 수평 분할선 드래그 → SQL/Plan 높이 비율 조절 (10~80%). All 스코프에서 활성
- h. 수직 분할선 드래그 → Before/After 좌우 비율 조절 (20~80%). SQL, Plan 스코프에서 활성
- i. Diff 색상 규칙: 변경=노란 배경(amber-100), 추가=초록 배경(emerald-100), 삭제=빨간 배경(red-100)

### (4) Describe Object 패널

**1. 내용**
- SQL 내 테이블/뷰 이름을 Ctrl+클릭하면 해당 오브젝트의 구조(컬럼 통계, 인덱스)를 우측 패널로 표시한다. 여러 오브젝트를 탭으로 동시 열람 가능.

**2. 구성**
- a. 탭 바 — 여러 오브젝트를 탭으로 관리. 각 탭에 타입 배지(`T`=테이블, `V`=뷰) + 오브젝트명 + X 닫기 버튼. 활성 탭은 하단 파란 보더
- b. 테이블 헤더 — `OWNER.TABLE_NAME`, 행 수, 블록 수, 평균 행 길이, 분석 시점. 파티션이면 파티션 타입·키·수 배지
- c. 컬럼 섹션 (상단, 리사이즈 가능)
 - 컬럼 테이블 — Name / Type / Null / Distinct / Nulls / Histogram / Len / Density / Bucket / Default / Comments
 - SQL에서 사용되는 컬럼 행 강조 (연한 배경)
 - 컬럼 클릭 → 색상 하이라이트 (인덱스 섹션에서 해당 컬럼 포함 행도 연동 강조)
- d. 컬럼/인덱스 분할선 — 드래그로 높이 비율 조절 (25~80%)
- e. 인덱스 섹션 (하단)
 - 인덱스 테이블 — Index Name / Type / Uniq / Columns / Status / BLv / Leaf / CF / DK
 - UNIQUE 인덱스는 초록 `UQ` 배지, Status VALID/INVALID 색상 표시
 - 인덱스 Columns의 각 컬럼명 클릭 → 컬럼 섹션과 연동 하이라이트
- f. 뷰일 때 — 소스 텍스트(접기/펼치기), 컬럼 매핑(View Column → Base Table.Column), 참조 테이블 버튼(클릭 → 새 Describe 탭)
- g. 패널 너비 — 기본 340px, 좌측 리사이즈 핸들로 220px~70% 뷰포트 범위 조절

**3. 동작**
- a. SQL 비교 영역에서 Ctrl+클릭(또는 F4) → 해당 오브젝트 Describe 패널 열기
- b. Describe 패널 열릴 때 → 상세 정보 패널 자동 접힘 (공간 확보)
- c. 모든 Describe 탭 닫기 → 상세 정보 패널 자동 복원
- d. Ctrl 키 누른 상태 → SQL 내 테이블/뷰 이름에 밑줄 + 파란색 + 포인터 커서 표시
- e. ESC → 모든 Describe 탭 일괄 닫기 (LIFO 스택의 한 단계)

### (5) 슬라이드 패널과의 차이 요약

| 항목 | T1 슬라이드 | T2 워크스페이스 |
|------|------------|----------------|
| SQL/Plan 비교 | 좁으면 Before/After 탭 전환 | 스코프 전환 (SQL/Plan/All/Bind) |
| 스코프 바 | 없음 | 있음 — 3~4종 스코프 + Format/Diff 토글 |
| 전체보기 | ComparePanel 플로팅 팝업 | 인라인 확대 (스코프 바에 라벨+닫기) |
| 상세 정보 배치 | 콘텐츠와 세로 스택 | 독립 좌측 패널 (접기/리사이즈) |
| 드래그 분할 | 없음 | SQL/Plan 수평 + Before/After 수직 분할선 |
| 복사 버튼 | 섹션별 1개 | Before/After 각 패널에 독립 복사 |
| Diff 범례 | 없음 | 변경(노랑)·추가(초록)·삭제(빨강) 칩 |
| 탭 라벨 (2번째 탭) | 권고안 | 반영 |
| 작업 목록 | 없음 (목록 페이지에서 클릭) | 좌측 사이드바 (260px, 접기 가능) |
| 상세검토 버튼 | 표시 (워크스페이스 진입) | 미표시 (이미 워크스페이스) |
| 승인완료 배너 버튼 | "권고안 탭으로 이동" | "반영 탭 →" |
| ESC | 패널 닫힘 | LIFO 스택 (전체보기→Describe→패널 닫기) |
| 액션 버튼 | 콘텐츠 하단 스티키 | 좌측 패널 하단 고정 |
| Describe Object | 없음 | SQL 내 오브젝트 클릭 → 우측 패널 |
| 새 입력 | 없음 | 헤더 "+ 새 입력" 버튼 |
